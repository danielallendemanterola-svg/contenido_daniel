Configurar tu agente
Para que la IA funcione necesitas una API key de Anthropic. Solo se configura una vez y se guarda en tu navegador.
  • 01Ve a console.anthropic.com y crea una cuenta gratis
  • 02En el menú izquierdo haz clic en API Keys
  • 03Haz clic en Create Key, copia la clave que empieza con sk-ant-
  • 04Pégala aquí abajo y haz clic en Guardar

Tu clave se guarda solo en tu navegador (localStorage). No se envía a ningún servidor externo excepto a Anthropic directamente. Costo estimado: ~$1–5 USD/mes con uso normal.
